Server type Recommended type of company Approximate time of embedding Total price
Basic type Small businesses 1-3 weeks ~1000$
Average server Medium size businesses 1 month ~10000$
Big server Big companies 2-4 months ~500000$

Basic type of server is a cheap and available decision for small businesses that want to improve their information management greatly. Basically it is a bunch of computers that hold information, connected to a main computer, that manages all the information

Average type of server is a normal server that alredy costs a lot but can operate a huge amount of information comparing to a basic type. Also it occupies a little more space - it will take a full room inside your office

Big server type can operate the biggest amount of information but it also holds enormous area. You will need at least a building to accomodate it. Also it costs a lot of money, but for big companies such upgrades promise much bigger income
